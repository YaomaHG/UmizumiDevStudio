# UmizumiDevStudio
Repositorio creado para la optativa de Programación de Videojuegos en 6to. Semestre de la carrera de Ingeniería de Software.

## Presentación del equipo

Somos un equipo de estudiantes de Ingeniería de Software apasionados por el desarrollo de videojuegos. Nuestro objetivo es explorar y aprender sobre el diseño, programación y producción de videojuegos, combinando nuestras habilidades técnicas con creatividad para crear experiencias interactivas significativas.

## Intereses generales

- Diseño de videojuegos y mecánicas de juego
- Programación de sistemas y herramientas para videojuegos
- Narrativa interactiva y diseño de experiencias
- Desarrollo de software y pensamiento lógico aplicado a videojuegos
- Experimentación con diferentes géneros y plataformas
